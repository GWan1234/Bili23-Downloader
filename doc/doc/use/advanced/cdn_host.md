# 替换音视频流 CDN 节点
由于哔哩哔哩（B 站）默认分配的 CDN 线路可能存在稳定性问题，导致音视频流下载失败，建议开启`替换音视频流 CDN 节点`功能。该功能会根据您设置的优先级顺序，自动选择可用的 CDN 节点，以提升访问速度和成功率。

<img src="https://s21.ax1x.com/2025/05/16/pEjXOGq.png" alt="自定义 CDN 节点" style="width: 90%;">

*图1 自定义 CDN 节点示意图*

列表中包含多个备用 CDN 节点，您可调整它们的优先级顺序。  
优先级规则 ：数值越小的节点优先级越高（例如优先级设为`1`的节点会比`3`的节点更优先被调用）。  
状态标注的`未检测`表示该节点尚未被系统主动探测，需通过`Ping 测试`确认实际连通性。

:::warning
开启代理工具时，请务必关闭此功能，避免 CDN 节点与代理线路冲突导致连接失败。
如果开启功能后仍出现下载失败，请关闭该功能。
:::
